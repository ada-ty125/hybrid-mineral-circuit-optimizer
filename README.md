## The ESEGraph library

This library is hosted at <https://github.com/ese-msc/esegraph> and is included as a git submodule in this repository. Submodules are a way to include one git repository inside another, while allowing each to be managed independently. You can find the code for the ESEGraph library in the `esegraph` directory of this repository.

To clone this repository along with the ESEGraph submodule, you can use the following command:

```bash
git clone --recursive <repository_url>
```

To update the ESEGraph library to the latest version (if you are told to), you can use the following commands in your terminal:

```bash
git submodule update --remote 
```

If your group discovers a bug in the ESEGraph library, you can report it by opening an issue on the ESEGraph GitHub repository. If you have a fix for the bug, you can fork the ESEGraph repository, make your changes, and then submit a pull request to the original repository.
